Project Name: AI-Powered Lead Response Automation
1. Problem Statement:
The task was to create an automated lead response workflow using
n8n that captures form submissions, sends AI-generated replies,
handles follow-ups, and stores lead information.
2. Solution Implemented:
Developed a frontend form page integrated with n8n workflow. The AI
reads customer form submissions and checks lead priority. High-intent
or urgent leads are directly forwarded to the admin. Normal leads
receive automated customized email responses and follow-up mails
after one day.
3. Technologies Used:
React.js, n8n, OpenAI API, Google Sheets, Email Automation
4. Issues / Blockages:
Faced issues while connecting HubSpot credentials and webhook
integration. Due to this, Google Sheets was used instead of HubSpot
for storing lead data.
5. Project Status:
Completed Successfully
